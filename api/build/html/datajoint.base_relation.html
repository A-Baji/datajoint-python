class datajoint.base_relation.BaseRelation(arg=None)[source]¶

Bases: datajoint.relational_operand.RelationalOperand

BaseRelation is an abstract class that represents a base relation, i.e. a table in the database. To make it a concrete class, override the abstract properties specifying the connection, table name, database, context, and definition. A Relation implements insert and delete methods in addition to inherited relational operators.

insert(rows, replace=False, ignore_errors=False, skip_duplicates=False)[source]¶

Insert a collection of rows.

Parameters:
  • rows – An iterable where an element is a numpy record, a dict-like object, or an ordered sequence. rows may also be another relation with the same heading.
  • replace – If True, replaces the existing tuple.
  • ignore_errors – If True, ignore errors: e.g. constraint violations.
  • skip_duplicates – If True, silently skip duplicate inserts.

Example:

>>> relation.insert([
>>>     dict(subject_id=7, species="mouse", date_of_birth="2014-09-01"),
>>>     dict(subject_id=8, species="mouse", date_of_birth="2014-09-02")])
